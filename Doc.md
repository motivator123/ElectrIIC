## Описание приложений проекта  


### main  
-Главная страница  
-Страницы хедера  
-Страницы футера  
-Переходы между страницами  


### website  
-Отвечает за переход на главную страницу (встречает пользователя и перенаправляет в main)  


### shop  
-Страница каталога  
-Страница товара  
-Описание моделей базы данных  
-Страница заказа товара
